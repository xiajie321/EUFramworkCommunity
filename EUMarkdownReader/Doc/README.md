# Markdown文档管理器

## 概述

Markdown文档管理器是一个基于UIToolkit的文档阅读工具，用于统一管理和浏览EUFramework框架中所有扩展的文档。

## 功能特性

- **自动扫描**：自动扫描所有已安装扩展的Doc文件夹
- **树形展示**：树形结构展示文档层级
- **实时渲染**：实时渲染Markdown内容
- **富文本支持**：支持标题、段落、列表、代码块、表格、图片、视频等常见Markdown语法
- **深色主题**：深色主题界面，与Unity编辑器风格一致
- **搜索功能**：支持文档内搜索（文件名/内容）

## 使用方法

### 打开文档阅读器

在Unity编辑器菜单栏中选择：

```
EUFramework -> 文档阅读器
```

### 浏览文档

1. 窗口左侧显示文档树，按扩展分组
2. 点击扩展名称可展开/折叠该扩展的文档列表
3. 点击具体的文档文件名，右侧将显示文档内容
4. 使用顶部的"刷新"按钮可重新扫描所有文档

### 文档组织规范

为了让文档能被正确识别和显示，请遵循以下规范：

1. 在扩展根目录下创建`Doc`文件夹
2. 将Markdown文档（.md文件）放入Doc文件夹
3. 可以在Doc文件夹下创建子文件夹来组织文档
4. 建议至少包含一个README.md作为主文档

### 目录结构示例

```
YourExtension/
├── extension.json
├── Doc/
│   ├── README.md          # 主文档
│   ├── QuickStart.md      # 快速开始
│   ├── API/               # API文档目录
│   │   ├── Core.md
│   │   └── Utils.md
│   └── Examples/          # 示例文档目录
│       └── BasicUsage.md
├── Script/
└── ConfigPanel/
```

## 支持的Markdown语法

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 段落与格式

- 普通文本段落，支持自动换行
- **粗体文本**
- *斜体文本*
- ~~删除线文本~~
- `行内代码`

### 列表

无序列表：
```markdown
- 项目1
- 项目2
- 项目3
```

有序列表：
```markdown
1. 第一项
2. 第二项
3. 第三项
```

任务列表：
```markdown
- [ ] 待办事项
- [x] 已完成
```

### 表格

支持标准 Markdown 表格语法，包括对齐方式（左对齐、居中、右对齐）：

```markdown
| 标题1 | 标题2 | 标题3 |
| :--- | :---: | ---: |
| 左对齐 | 居中 | 右对齐 |
| 内容 | 内容 | 内容 |
```

### 代码块

使用三个反引号包裹代码：

```csharp
public class Example
{
    public void Method()
    {
        Debug.Log("Hello World");
    }
}
```

支持指定语言类型，如：csharp, javascript, python, json 等。

### 链接与多媒体

- 链接：`[链接文本](URL)`
- 图片：`![Alt文本](图片路径)` (支持本地相对路径和网络URL)
- 视频：直接链接到视频文件（.mp4, .mov等）将自动显示为播放器
