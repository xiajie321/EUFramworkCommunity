# 版本更新日志

## [1.1.3] - 2026-03-02

### 优化

- 全面优化文档阅读器的视觉设计，提升阅读舒适度，降低视觉疲劳
- 调整标题、段落、列表的排版间距，使文档层次更加分明
- 优化引用块、代码块和表格的颜色对比度，使其更柔和
- 改进导航栏和侧边栏的视觉层次和交互状态颜色
- 修复了 USS 样式表中的警告信息
- 拓展名称由EUMarkdownDocManager更改为EUMarkdownReader
